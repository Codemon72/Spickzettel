### Shortcuts (for Mac)

- `CMD + J` - open terminal
- `CMD + P` - open file (start typing name...)
- `ALT + CMD + (up or down)` - expand cursor over more lines up or down
- `SHIFT + ALT + (up or down)` - copy whole line up or down
- `SHIFT + CMD + 7` - comment out line
